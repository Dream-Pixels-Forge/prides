# New Feature Development

**Feature Request:** {{args}}

## Workflow

0. Call `@prides-agent` to start the new feature implementation workflow
1. Brainstorm and refine the feature concept
2. Generate product requirements document if needed
3. Break down into actionable tasks
4. Create feature branch

5. Design specification and user experience if needed
6. Create visual prototype (Stitch integration) if needed

7. Implement the feature
8. Code quality check (apply auto-fixes)
9. Deep code review for issues

10. Run tests
11. Performance validation
12. WCAG compliance check

13. Execute deployment strategy
