# Release v0.4.0

**Date:** 2026-03-27

## Summary

Minor release introducing comprehensive PRIDES.md documentation system, enhanced skills awareness, and extensive audit fixes across all agent files. This release focuses on improving documentation quality, agent coordination, and system maintainability.

---

## Key Features

### 1. PRIDES.md Documentation System

New comprehensive documentation structure for project context and decision tracking:

| File | Purpose |
|------|---------|
| **project-context.md** | Central project information and architecture overview |
| **agent-coordination.md** | Agent workflow patterns and coordination protocols |
| **active-workflow.md** | Current active workflow state and progress tracking |
| **decision-log.md** | Architectural and implementation decision records |
| **quality-gates.md** | Quality standards and enforcement mechanisms |
| **skills-inventory.md** | Complete skills catalog and usage guidelines |

### 2. Enhanced Skills Awareness

- Improved skills discovery and integration
- Better skills documentation in SKILLS-INDEX.md
- Enhanced skills-to-agents mapping
- Updated skills usage patterns across all agents

### 3. Comprehensive Audit Fixes

- Fixed version metadata consistency across all agent files
- Updated YAML frontmatter to latest standards
- Standardized lastUpdated dates across all files
- Improved agent descriptions for better triggering accuracy
